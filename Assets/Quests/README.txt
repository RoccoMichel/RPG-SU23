To Create a new Quest / Quest Element: Open the Context menu within the assets folder and navigate to 'Quests' at the lower end.
From there Select: 'Dialog Element', 'Travel Element', 'Slaughter Element', 'Fetch Element' or 'Quest'.

EACH QUEST AND ITS ELEMENTS SHOULD BE PUT IN THEIR OWN DESIGNATED FOLDER WITHIN Assets>Quests
FOLDER ARE NAMED "QUEST [order of appearance]"
THE QUEST FILE ITSELF SHOULD HAVE A SIMPLE NAME HITTING TO WHAT IT CONTAINS (should be easy to find when inspecting a quest giver)

QUEST ELEMENTS ARE NAMED IN THE FOLLOWING MANNER:
- QUEST NUMBER INDEX
- '-'
- INDEX OF ELEMENT ORDER
- ' '
- NAME OF TYPE (Dialog, Travel, Slaughter, Fetch)

NUMBERING STARTS AT 1!
EXAMPLE: Fetch 3-4 (the quest element of the third quests 4th stage
